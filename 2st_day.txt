1. CSS(Cascading Style Sheet)
  1.1 배경
     HTML( 웹 문서의 내용 ) + 디자인 요소를 분리 => 웹 표준
  1.2 스타일(style)
     HTML 문서에서 자주 사용하는 글꼴, 색상, 정렬, 각 요소의 배치 방법
     과 같이 문서의 겉모습을 결정짓는 것.
  
     반응형 웹 디자인 : 사용자가 PC로 접속을 하든, 모바일로 접속을 하든
     웹 브라우저의 크기에 따라 화면의 레이아웃을 자동으로 바꿔주는 방법.

2. 스타일과 스트일 쉬트
  2.1 분류
     스타일 시트 -- 브라우저 기본 스타일
                     웹 브라우저에서 웹 문서를 표시할 때 브라우저에서 기본으로 사용하는 스타일
                -- 사용자 스타일
                    - 인라인 스타일
                    - id 스타일
                    - class 스타일
                    - 타입 스타일
                    - 내부 스타일
                    - 외부 스타일
  2.2 기본 형식
     선택자 { 속성1:속성값 ; 속성2:속성값 ; }

     p { text-align : center ; color : blue ; }

     주석 : /* */

     CSS 경량화 : CSS 소스의 주석, 줄 바꿈, 공백 등을 제거하여 필요한 정보만
     남겨서 파일을 작게 만드는 것. css minify, css compress 로 검색해서 사용.


  2.3 div 태그와 span 태그
   - 용도
     웹 페이지를 만들 때 관련있는 요소끼리 그룹으로 묶으면 레이아웃을
     구성하기 쉬워지고, HTML 구조를 쉽게 작성할 수 있음.
   - 차이점
     div :  블록 요소를 그룹으로 묶을 때 사용.
     span : 인라인 요소를 그룹으로 묶을 때 사용.
            블록 요소를 묶을 수 없음.

  2.4 선택자
   - 전체 선택자
     문서의 모든 요소에 스타일을 적용.
   - 타입 선택자
     특정 태그를 사용한 모든 요소에서 스타일을 적용.
   - 클래스 선택자
     특정 부분만 선택해서 문서 안에 여러 번 적용. 마침표를 붙여서 사용
   - id 선택자
     특정 부분만 선택해서 문서 안에서 한 번만 적용. #을 붙여서 사용
   - 그룹 선택자
     여러 요소에 같은 스타일을 적용할 때 사용.
     h1, h2 { text-align : center ; }

  2.5 캐스캐이딩(Cascading)
   - 의미
     CSS에서 웹 요소에 둘 이상의 스타일을 적용시, 우선순위에 따라 적용할
     스타일을 결정하게 됨.
   - 캐스캐이딩 - 스타일 우선순위
      1순위 : !important
      2순위 : 인라인 스타일, inline.html 참조
      3순위 : id 스타일, id.html 참조
      4순위 : class 스타일, class.html 참조
      5순위 : 타입 스타일, type.html 참조

   - 캐스캐이딩 - 스타일 상속
      스타일 시트에서 자식 요소에서 별도로 스타일을 지정하지 않으면 부모 요소의
      스타일 속성들이 자식 요소로 전달되는 것.

      예 : html에서 body tag는 웹 문서에서 사용한 모든 tag 의 부모 요소임.
           body tag 에 글자색, 글꼴을 지정하면, 웹 문서 전체에 적용됨.

           자식 요소에 스타일을 적용하지 않으면, 부모(body tag) 에 적용된
           글자색, 글꼴을 사용하게 됨.


2. 텍스트 관련 스타일
  2.1 글꼴 관련 스타일
      - 글꼴 스타일
        글자의 모양새를 지정
      - 문단 스타일
        웹 문서에 표시되는 텍스트를 지정
  2.2 속성
      아래의 속성이외에소 많음.
      - font-family : 글꼴 지정
        body { font-family : "맑은 고딕", 돋움, 굴림 }
        스타일 상속을 고려.
      - font-size : 글자 크기
        단위 : px, pt, em(많이 사용 됨.)
        em : 부모 요소에서 지정한 글꼴의 대문자 M의 너비를 기분으로 한 후 
            비율값으로 지정. 1em은 16px, 12pt임.
      - font-style : 이탤릭체로 표시
      - font-weight : 글자 굵기
        normal, bold, bolder, lighter, 100(가장 가늘게) ~ 900(가장 굵게)
  2.3 웹 폰트 - 구글 폰트 사용
      - 주소 : fonts.google.com
  2.4 텍스트 관련 스타일 - 글자색 지정
      - 16진수
        형식 : #RRGGBB
        범위 : #000000(black) ~ #ffffff(white)
      - hsl, hsla
        hsl : h(색상), s(채도),l(명도)
        hsla : h(색상), s(채도),l(명도), a(불투명도)
        사용 : hsl(0, 100%, 50%)
               hsla(0, 100%, 50%, 0.5)
      - 영문명으로 지정하면
        white, black, red, blue
      - 정렬
        속성명 : text-align
        속성값 : start, end, left, right, center, justify(양쪽 맞춤) 등
      - 그림자 효과
        속성명 : text-shadow
        속성값 :
          가로거리 : 글자로부터 그림자까지의 가로 거리
                    양수=>글자의 오른쪽, 음수=>글자의 왼쪽에 그림자를 나타냄.
          세로거리 : 글자로부터 그림자까지의 세로 거리
                    양수=>글자의 아래쪽, 음수=>글자의 위쪽에 그림자를 나타냄.
          번짐정도 : 양수=>모든 방향으로 퍼짐. 음수=>모든 방향으로 축소.
          색상 : 그림자 색상
  2.5 표 스타일
      - 표 제목 위치 설정 : cation-side
      - 표 테두리 : border
      - 셀 여백 : border-spacing
      - 표, 셀 합치는 것 : border-collapse
