1. CSS(Cascading Style Sheet)
  1.1 배경
     HTML( 웹 문서의 내용 ) + 디자인 요소를 분리 => 웹 표준
  1.2 스타일(style)
     HTML 문서에서 자주 사용하는 글꼴, 색상, 정렬, 각 요소의 배치 방법
     과 같이 문서의 겉모습을 결정짓는 것.
  
     반응형 웹 디자인 : 사용자가 PC로 접속을 하든, 모바일로 접속을 하든
     웹 브라우저의 크기에 따라 화면의 레이아웃을 자동으로 바꿔주는 방법.

2. 스타일과 스트일 쉬트
  2.1 분류
     스타일 시트 -- 브라우저 기본 스타일
                -- 사용자 스타일
                    - 인라인 스타일
                    - id 스타일
                    - class 스타일
                    - 타입 스타일
                    - 내부 스타일
                    - 외부 스타일
  2.2 기본 형식
     선택자 { 속성1:속성값 ; 속성2:속성값 ; }

     p { text-align : center ; color : blue ; }

     주석 : /* */

     CSS 경량화 : CSS 소스의 주석, 줄 바꿈, 공백 등을 제거하여 필요한 정보만
     남겨서 파일을 작게 만드는 것. css minify, css compress 로 검색해서 사용.
