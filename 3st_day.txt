1. CSS 박스 모델
  1.1 개요
    - 웹 문서에서 내용을 배치할 때 각 요소를 박스 형태로 구성함.
    - CSS를 사용한 레이아웃의 기본이 됨. => 중요함.
    - 이런 박스 모델이 모여서 웹 문서가 됨.

  1.2 박스 모델의 요소
    - 블록 레벨 요소
      태그를 사용해서 요소를 삽일했을 때 혼자 한 줄을 차지하는 것
      <h1>, <div>, <p>
    - 인라인 레벨 요소
      한 줄을 차지하지 않음. 콘텐츠 만큼만 영역을 차지함.
      <span>,<img>,<strong>  

  1.3 박스 모델의 기본 구성
    - 콘텐츠 영역
      크기 지정 속성 : width, height

    - 패딩(padding) : 박스와 콘테츠 영역 사이의 여백
    - 테두리(border) : 박스의 테두리
    - 마진(margin) : 여러 박스 모델들 사이의 여백

    - 그림자 효과
      속성명 : box-shadow
      속성값 : 
        수평거리 : 그림자가 가로로 얼마나 떨어져 있는지 정도.
                양수는 요소의 오른쪽, 음수는 요소의 왼쪽
        수직거리 : 그림자가 세로로 얼마나 떨어져 있는지 정도.
                양수는 요소의 아래쪽, 음수는 요소의 위쪽.
        흐림정도 : 0 이 기본값이고 진한 그림자를 표시.
                숫자가 크질수록 부드러운 그림자를 표시. 음수값 사용 불가.
        색상 : 한 가지 색상만 지정 가능. 기본값은 검은색.
        insert : 이 키워드를 함께 사용하면 안쪽으로 그림자를 표시.

    - 테두리 스타일
      속성명 : border-style
      속성값 :
        none : 테두리가 없음. 기본값.
        solid : 실선
        dotted : 점선
        dashed : 짧은 직선
        double : 이중선. 이중선의 간격조절은 border-width 로 조정.
        이외에 다양하게 있음.

    - 마진의 제어 및 활용
      두 박스 모델 사이의 여백을 마진이라 함.
      한 박스 모델에서의 여백은 패딩.

      속성명 : margin
      속성값 : px, 백분율, 
              auto 속성에서 지정한 값에 맞게 자동으로 지정.

    - 마진 중첩(마진 상쇄)
      박스 모델 요소를 세로로 배치할 경우에 각 요소의 마진과 마진이
      서로 만나면 마진값이 큰 쪽으로 겹치는 문제가 발생하는 것.

      오른쪽 마진과 왼쪽 마진이 만나는 경우는 중첩되지 않음.

    - 패딩
      콘텐츠 영역과 테두리 사이의 여백임.

  1.4 레이아웃 만들기
    - 배치 방법 결정
      속성명 : display
              블록 레벨 요소와 인라인 레벨 요소를 서로 바꿔서 사용할 수 
              있도록 함.

              네비게이션을 만들면서 메뉴항목을 가로로 배치할 경우에 많이 사용.
      속성값 : 
          block : 인라인 레벨 요소를 블록 레벨 요소로 만듬.
          inline : 블록 레벨 요소를 인라인 레벨 요소로 만듬.
          inline-block : 인라인 레벨 요소와 블록 레벨 요소의 속성을 모두 가짐.
          none : 해당 요소를 화면에 표시하지 않음. 기본값임.

    - 이미지와 텍스트 동시 배치
      속성명 : float
              float 속성을 사용하면 이미지를 표시하고 그 주변에 텍스트를
              배치할 수 있음.
      속성값 : left, right, none
      
      float를 해제하려는 경우.
      속성명 : clear
      속성값 : left, right, both

  1.5 웹 요소의 위치지정
      속성명1 : position
      속성값 : static : html 문서의 흐름. html tag 작성 순서.
              relative : static 동일. 추가적으로 위치값을 지정 가능.
              absolute : relative 를 적용한 상위 요소를 기준으로 위치를 지정.
              fixed : 브라우저 기준으로 위치를 지정.

      속성명2 :
              left : 기준 위치와 요소 사이에 왼쪽으로 얼마나 떨어져 있는지 지정. 
              right : 기준 위치와 요소 사이에 오른쪽으로 얼마나 떨어져 있는지 지정. 
              top : 기준 위치와 요소 사이에 위쪽으로 얼마나 떨어져 있는지 지정. 
              bottom : 기준 위치와 요소 사이에 아래쪽으로 얼마나 떨어져 있는지 지정. 

      사용예 : 
          position : absolute;
          left : 50px;
