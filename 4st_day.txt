1. 고급 선택자
  1.1 연결 선택자
    - 둘 이상의 선택자를 연결해서 사용.
    - 조합 선택자, 콤비네이션 선택자, 콤비네이션 셀렉터

    1.1.1 하위 선택자
      - 부모 요소에 포함된 하위 요소를 선택해서 사용.
      - 사용예
        section p { ... }

    1.1.2 자식 선택자
      - 하위 선택자와 다르게 자식 요소에만 스타일을 적용하는 선택자.
      - 사용예
        section > p { ... }

    1.1.3 인접 형제 선택자 ( 인접의 의미가 중요. )
      - 형제 요소중에서 첫 번째 동생 요소만 선택함.
      - 사용예
        h1 + p { ... }
        h1의 인접한 p의 형제 요소중에서 가장 인접한 p 요소를 선택.

    1.1.4 형제 선택자
      - 인접 형제 선택자와 달리 모든 형제 요소에 적용.
      - 사용예
        h1 ~ p { ... }


  1.2 속성 선택자
    1.2.1 특정 속성이 있는 요소를 선택
      - 사용예
        a[href] { ... }
        a태그 중에서 링크가 있는 요소만 선택

    1.2.2 특정 속성값이 있는 요소를 선택
        a[targer = _blank] { ... }
        새탭으로 열리는 링크가 있는 요소를 선택

    1.2.3 여러 값 중에서 특정 속성값이 포함된 속성 요소를 선택
        [class ~= button] { ... }
        속성의 값 중에서 button 으로 값이 설정된 요소를 선택


    1.2.4 특정 속성값이 포함된 속성 요소를 선택
    1.2.5 특정 속성값으로 시작하는 속성 요소를 선택
    1.2.6 특정 값으로 끝나는 속성의 요소를 선택
    1.2.7 일부 속성값이 일치하는 요소를 선택

  1.3 가상 클래스와 가상 요소
    1.3.1 가상 클래스
      1.3.1.1 사용자 동작에 반응하는 가상 클래스
        - ":link"
          방문하지 않은 링크에 스타일 적용
          아직 링크를 클릭하지 않은 경우
        - ":visited"
          방문한 링크에 스타일 적용
          링크를 클릭했는 경우
        - ":hover"
          특정 요소에 마우스 포인트를 올려놓는 경우
        - ":active"
          웹 요소를 활성화 한 경우
          클릭한 경우
        - ":focus"
          웹 요소에 초점이 맞추어진 경우
          초점이 맞추어진 경우

      1.3.1.2 요소 상태에 따른 가상 클래스
        - ":target"
          앵커 대상에 스타일을 적용
        - ":enabled", ":disabled"
          해당 요소가 사용할 수 있을 때 : ":enabled"
          해당 요소가 사용할 수 없을 때 : ":disabled"
        - ":checked"
          선택한 항목의 스타일을 적용
          예로 폼의 라이오 박스나 체크 박스에서 선택된 항목에
          스타일을 적용할 때
        - ":not"
          특정 요소를 제외하고 스타일을 적용

      1.3.1.3 구조 가상 클래스
        특정 요소에 스타일을 적용시 class 또는 id 식별자를 이용하지만,
        단점이 개별 적용하는 것으로 됨. 해당 요소중 몇 번째만 적용. 안됨!!

        class, id 를 사용하지 않고, 지정할 요소가 몇 번째인지를 선택해서
        스타일을 적용할 수 있음.

        :only-child
          부모 안에 자식 요소가 하나뿐일 때 자식요소를 선택.
        A:only-type-of
          부모 안에 A 요소가 하나뿐일 때 선택
        :first-child
          부모 안에 있는 모든 요소 중에서 첫 번째 자식 요소를 선택.
        :last-child
          부모 안에 있는 모든 요소 중에서 마지막 자식 요소를 선택.
        A:first-of-type
          부모 안에 있는 A 요소 중에서 첫 번째 요소를 선택.
        A:last-of-type
          부모 안에 있는 A 요소 중에서 마지막 요소를 선택.
        :nth-child(n)
          부모 안에 있는 모든 요소 중에서 n 번째 자식 요소를 선택.
        :nth-last-child(n)
          부모 안에 있는 모든 요소 중에서 끝에서 n 번째 자식 요소를 선택.
        A:nth-of-type(n)
          부모 안에 있는 A 요소 중에서 n번째 요소를 선택
        A:nth-last-of-type(n)
          부모 안에 있는 A 요소 중에서 끝에서 n번째 요소를 선택

      1.3.1.4 가상 요소
        가상 클래스가 웹 문서의 여러 요소 중에서 원하는 요소를 선택한다면,
        가상 요소는 문서 안의 특정 부분에 스타일을 지정하기 위해 가상 요소를
        만들어 추가해서 사용함.

        가상 클래스와 구별할 수 있도록 콜론 2개를 붙여서 표시함.

        - "::first-line 요소", "::first-letter 요소"
          첫 번째 줄, 첫 번째 글자에 스타일을 적용.

        - "::before 요소", "::after 요소"
          내용 앞뒤에 콘텐츠를 추가
